# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px 
- Desktop: 1360px


## Colors

- white: hsl(0%, 0%, 100%)
- Black: hsl(22.5, 16.67%, 9.41%)

## Typography

### Body Copy

- Font size (title): 32px
- Font size (subtitle): 20px
- Font size (paragraph): 14px
